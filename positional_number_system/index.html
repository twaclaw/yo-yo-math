3.2 Positional number system

TipThe Babylonians

The Babylonians, who developed the sexagesimal system that we still use to tell the time, were the first to develop a positional number system.

The Babilonians had discovered that the question “How many?” can be enormously simplified if the same number stands for different values depending on its position in a row of numbers. Place-value notation eliminated the need to invent new digits for numbers, no matter how large. A Contextual History of Mathematics, Ronald Calinger (Prentice Hall, 1999)

This is a significant development because it means that no matter how large a number is, it can always be represented with just ten symbols.

Numbers greater than \(9\) will be represented by combinations of the same ten digits in different positions. Any combination that starts with a number other than \(0\) is a valid number.

\[ 10, \quad 123, \quad 9999999, \quad 100000000000, \quad \ldots \]

TipPositional number system in a nutshell

The defining feature of the positional number system is that the value of a digit depends on its position in the number. For instance, in the number \(333\), each digit \(3\) has a different value.

The farther to the left \(\Leftarrow\) a digit is, the larger its value. We will use boxes to emphasize the positions.

A digit’s value changes as soon as it is moved to a different position. For example, the resulting number changes each time the digit \(3\) is moved to the left; the larger the position to the left, the larger the number.

However, in the last example, these are not valid numbers. We need to fill the empty positions to the right.

  • \(\text{\style{font-size:1.5em}{👮‍♀️}}\) Any positions to the right of a digit must be filled with additional digits. For the time being, let’s use zeros, but any digits will do.

You don’t need to know what the numbers are; you only need to know that the further to the left the digit \(3\) is, the larger the value it represents1:

\[ {\color{olive}{3}}000 > {\color{blue}{3}}00 > {\color{red}{3}}0 > 3 \]

Let’s quickly recap what we just learned. The following are some rules \(\text{\style{font-size:1.5em}{👮‍♀️}}\) of the positional number system:

  • \(\text{\style{font-size:1.5em}{👮‍♀️}}\) The first position is the one on the far right. Values increase as we move to the left.

  • \(\text{\style{font-size:1.5em}{👮‍♀️}}\) Any positions to the left of the number that are not used are left empty2.

  • \(\text{\style{font-size:1.5em}{👮‍♀️}}\) The same digit can represent different values depending on where it is in a number. The further to the left the digit is, the larger the value it represents.

  • \(\text{\style{font-size:1.5em}{👮‍♀️}}\) Any positions to the right of a digit must be filled with additional digits.

3.5 Base 10

The base of the decimal system is \(10\).

The reason the number system we have used so far is called the “decimal number system” and each floor of our Grand Hotel has ten rooms is because we are using a number system based precisely on the number \(10\). Ten is the base of this system. What does that mean?

On the one hand, it means that we have ten digits or symbols:

The decimal numbers. Ten digits.

It also means that all numbers greater than \(9\) are constructed adding one of these ten digits to combinations of \(10\)3.

In our Grand Hotel analogy, climbing a floor means adding up \(10\). Each time we climb a ladder, we add \(10\). If we climb two ladders we add two times \(10\), \(2\times 10 = 10 + 10 = 20\), if we climb \(5\) ladders we add five times \(10\), \(5\times 10 = 10 + 10 + 10 + 10 + 10 = 50\), and so on.

This is how some of the new numbers we just learned are constructed using the base \(10\):

\[\begin{align*} 13 &= 3 + (1\times 10) = 3 + (\underbrace{10}_{1\text{ time}}) &= 3 + 10 \\ 23 &= 3 + (2\times 10) = 3 + (\underbrace{10 + 10}_{2\text{ times}}) &= 3 + 20 \\ 24 &= 4 + (2\times 10) = 4 + (\underbrace{10 + 10}_{2\text{ times}}) &= 4 + 20 \\ 35 &= 5 + (3\times 10) = 5 + (\underbrace{10 + 10 + 10}_{3\text{ times}}) &= 5 + 30 \\ \end{align*}\]
